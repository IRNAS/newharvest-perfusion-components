# New Harvest perfusion components

This repository contains design files for SLA/DLP manufactured components, which can be used for scaffold incubation in perfusion, as well as filtration of culture media during cell culture in perfusion (to remove tubing particulate abraded during peristalsis).

We are available for discussion via [e-mail](mailto:bostjan@irnas.eu).
